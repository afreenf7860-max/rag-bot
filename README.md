# RAG Bot

AI-powered Document Q&A Bot using:
- LangChain
- ChromaDB
- HuggingFace Embeddings

## Features
- Document ingestion
- Vector storage
- Query-based retrieval

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run ingestion:
   python ingest.py

3. Run query:
   python query.py
